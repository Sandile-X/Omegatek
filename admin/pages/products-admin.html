How to Edit Products

Option 1: JSON File

  1. Click "Download JSON" to get current products.json
  2. Open in text editor (VS Code, Notepad++, etc.)
  3. Edit product details (name, price, description, etc.)
  4. Validate JSON syntax at jsonlint.com
  5. Upload the modified file using form above

Option 2: CSV Import

  1. Click "Export CSV" to get current products
  2. Open in Excel or Google Sheets
  3. Edit products in spreadsheet
  4. Save as CSV file
  5. Convert CSV to JSON using provided script
  6. Upload JSON file
